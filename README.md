# 📚 Book Store App

A full-stack Book Store application built with React, Tailwind CSS, Node.js, Express.js, and MongoDB.

## Features

- User Authentication
- Admin Dashboard
- Cart & Wishlist
- Order Management
- Stripe Payment Integration
- AI Book Recommendations
- Revenue Analytics
- Top Selling Books Analytics
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB

### Other
- JWT
- Stripe
- OpenRouter AI
